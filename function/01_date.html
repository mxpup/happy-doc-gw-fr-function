# 日期函数

# 语法

// 函数名支持驼峰和下划线方式
GwDate(函数名或编码, 参数1, 参数2, ...)

# 格式化

函数名编码描述
format100自定义格式化，并内置了一些格式
format_date101格式化日期
format_time102格式化时间
format_date_time103格式化日期时间

# format

参数类型描述
1Date | String
2String格式化
  • 内置格式
编码描述
1yyyy-MM-dd
2HH:mm:ss
3yyyy-MM-dd HH:mm:ss
4yyyy 年 MM 月 dd 日
5HH 时 mm 分 ss 秒
6yyyy 年 MM 月 dd 日 HH 时 mm 分 ss 秒

# 示例

// 2021年6月8日
GwDate("format", "2021-06-08 23:23:59", "yyyy年M月d日");

// 2021-6
GwDate(100, "2021-06", "yyyy-M");

// 2021年06月08日
GwDate(100, "2021-06-08 23:23:59", 4);

DateTime parse = DateTool.parse("2021-06-08");
// 2021年06月08日00时00分00秒
GwDate(100, parse, 6);

DateTime parse = DateTool.parse("2021-06", "yyyy-MM");
// 2021年06月08日00时00分00秒
GwDate(100, parse, 6);

# format_date

参数类型描述
1Date 或 String需要格式化的日期

# 示例

// 2021-06-08
GwDate('format_date', '2021-06-08 23:23:59');

DateTime parse = DateTool.parse("2021-06-08");
// 2021-06-08
GwDate(101, parse);

# format_date_time

# offset_day

参数类型描述
1Date 或 String需要偏移的日期
2Integer偏移的天数，正数往未来偏移，负数往历史偏移
// 2021-06-10 23:23:59
GwDate("offset_day", "2021-06-08 23:23:59",2);

DateTime parse = DateTool.parse("2021-06-08");
// 2021-06-06 00:00:00
GwDate(200, parse,-2);

函数名编码描述
is_eq300比较两个日期是否相等
is_eq_day301比较两个日期的天数是否相等
is_eq_month302比较两个日期的月数是否相等
is_eq_year303比较两个日期的年数是否相等
is_eq_year_month304比较两个日期的年月是否相等
is_eq_date305比较两个日期的年月日是否相等
is_eq_time306比较两个日期的时分秒是否相等
is_eq_date_time307比较两个日期的年月日时分秒是否相等
is_gt320第一个日期是否大于第二个日期
is_gte321第一个日期是否大于或等于第二个日期

# is_eq

参数类型必填描述
1Date 或 String是日期
2Date 或 String是日期
3String否将日期转为此格式，再进行比较
// false
GwDate("is_eq", "2021-06-08 23:23:59", "2021-12-08 23:23:59");

// true
GwDate(300, "2021-06-08 10:10:10", "2021-06-08 10:10:10");

// true
GwDate("IsEq", "2021-06-08 23:23:10", "2021-12-08 23:23:59", "HH:mm");

// true
GwDate(300, "2021-06-08 10:10:10", "2021-06-08 10:10:10");
