# 日期函数

# 语法

// 函数名支持驼峰和下划线方式
GwDate(函数名或编码, 参数1, 参数2, ...)

# 格式化

函数名编码描述
format100自定义格式化，并内置了一些格式
format_date101格式化日期
format_time102格式化时间
format_date_time103格式化日期时间

# format

参数类型描述
1date日期
2string格式化
  • 内置格式
编码描述
1yyyy-MM-dd
2HH:mm:ss
3yyyy-MM-dd HH:mm:ss
4yyyy 年 MM 月 dd 日
5HH 时 mm 分 ss 秒
6yyyy 年 MM 月 dd 日 HH 时 mm 分 ss 秒

# 示例

// 2021年6月8日
GwDate("format", "2021-06-08 23:23:59", "yyyy年M月d日");

// 2021-6
GwDate(100, "2021-06", "yyyy-M");

// 2021年06月08日
GwDate(100, "2021-06-08 23:23:59", 4);

DateTime parse = DateTool.parse("2021-06-08");
// 2021年06月08日00时00分00秒
GwDate(100, parse, 6);

DateTime parse = DateTool.parse("2021-06", "yyyy-MM");
// 2021年06月08日00时00分00秒
GwDate(100, parse, 6);

# format_date

参数类型描述
1date需要格式化的日期

# 示例

// 2021-06-08

// 2021-06-08
GwDate(101, parse);

# offset_day

参数类型描述
1date需要偏移的日期
2integer偏移的天数，正数往未来偏移，负数往历史偏移

DateTime parse = DateTool.parse("2021-06-08");
// 2021-06-06 00:00:00
GwDate(200, parse,-2);

# is_eq

参数类型必填描述
1date是日期
2date是日期
3string否将日期转为此格式，再进行比较
// false
GwDate('is_eq', '2021-06-08 23:23:59', '2021-12-08 23:23:59');

// true
GwDate(300, '2021-06-08 10:10:10', '2021-06-08 10:10:10');

// true
GwDate('IsEq', '2021-06-08 23:23:10', '2021-12-08 23:23:59', 'HH:mm');

// true
GwDate(300, '2021-06-08 10:10:10', '2021-06-08 10:10:10');

# 日期间隔

函数名编码描述

between_minute402两个日期之间的间隔分钟数
between_hour403两个日期之间的间隔小时数
between_day404两个日期之间的间隔天数
between_month405两个日期之间的间隔月数
between_year406两个日期之间的间隔年数
between_day_of_month_begin407日期到当月初之间的间隔天数
between_day_of_month_end408日期到当月末之间的间隔天数

# between_millisecond

参数类型必填描述
1date是开始日期
2date是结束日期
3boolean否是否忽略时分秒，默认 false
// 86400000

// 1000
GwDate(400, '2021-06-08', '2021-06-08 00:00:01');

// 0
GwDate(400, '2021-06-08', '2021-06-08 12:12:12', true);

1date是开始日期
2date是结束日期
3boolean否是否忽略时分秒，默认 true
// 1
GwDate('BetweenDay', '2021-06-08', '2021-06-09');

// 30
GwDate(404, '2021-06-08', '2021-07-08 01:01:01');

// 0
GwDate(404, '2021-06-08 00:00:02', '2021-06-09 00:00:01', false);

// 0
GwDate(404, '2021-06-08', '2021-06-08 12:12:12');

# 日期范围

函数名编码描述
range_month_begin_of_year501根据年份，返回每个月的月初日期
range_month_end_of_year502根据年份，返回每个月的月末日期

# range_month_begin_of_year

根据年份，返回每个月的月初日期

参数类型必填描述
1integer 或 string是年份
2string否日期格式化
// [2021-01-31,2021-02-28,2021-03-31,2021-04-30,2021-05-31,2021-06-30,2021-07-31,2021-08-31,2021-09-30,2021-10-31,2021-11-30,2021-12-31]
GwDate('range_month_begin_of_year', 2021);

// [2021年01月31日,2021年02月28日,2021年03月31日,2021年04月30日,2021年05月31日,2021年06月30日,2021年07月31日,2021年08月31日,2021年09月30日,2021年10月31日,2021年11月30日,2021年12月31日]
GwDate(501, '2021', 'yyyy年MM月dd日');
