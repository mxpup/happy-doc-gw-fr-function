# 字符串是否为空

函数名编码描述
is_blank100判断字符串是否为空
is_blank_all101判断字符串是否都为空
is_blank_any102判断字符串是否一个为空

# is_blank

参数类型描述
1any
// true
GwString('is_blank', '');

// true
GwString('IsBlank', ' ');

// true
GwString('isBlank', null);

// false
GwString(100, 0);

// false
GwString(100, false);

# is_blank_all

参数类型描述
1any
Nany
// true
GwString('is_blank_all', '', ' ', '   ', null);

// false
GwString('IsBlankAll', ' ', 'happy');

// false
GwString(101, 0, false);
